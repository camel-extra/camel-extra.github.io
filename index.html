Welcome to Camel Extra Component.

This project contains a number of extension components for the Apache Camel integration framework which due to GPL/LGPL licensing cannot be hosted at Apache.

Supported Components

  • camel-couchbase for working with the Couchbase document oriented NoSQL database
  • camel-db4o for working with the db4o NoSQL database
  • camel-esper for working with the Esper Event Stream Processing library
  • camel-exist for marshalling XML into the eXist database and querying it.
  • camel-hibernate for working with the Hibernate object relational mapping library
  • camel-jboss for running Camel inside JBoss Application Server 5.x.
  • camel-jboss6 for running Camel inside JBoss Application Server 6.x.
  • camel-jcifs for interacting with CIFS/SMB networking protocol via the JCIFS client
  • camel-virtualbox for interacting with VirtualBox via the web service API
  • camel-vtdxml for fast and efficient XPath evaluation using the VTD-XML library
  • camel-rcode for integration of the statistics environment R via Rserve
  • camel-spring-neo4j for interaction with Neo4j.
  • camel-zeromq for consuming or producing messages using ZeroMQ.
